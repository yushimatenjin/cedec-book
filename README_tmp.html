まえがき

近年 WebxXR などの流行り始めてきています。そんな中で WebGL ベースのゲームエンジンである PlayCanvas の認知度が徐々にあがりつつあります。WebGL を使用しゲームを作ることが出来るゲームエンジンですが、現在はまだゲームエンジニアがウェブでゲームを作ること、ウェブエンジニアがゲームを作ることへの隔たりがまだまだ大きいのも現状です。JavaScript のゲームエンジンである、PlayCanvas を使用することで少しでもその溝を解消できるかもしれません。本書については数ヶ月 PlayCanvas を触って得た知見や、ゲーム制作者からしたら常識かもしれないことであるが、もともとウェブ開発者であった筆者がゲームエンジンを使用した際に知らなかった部分などについては重点的に書かせていただきました。

本書では全体を通して、PlayCanvas を使用し、ゲーム開発をする為の一連の開発の流れを実際に手を動かしながら学ぶことができます。

1 章では PlayCanvas と PlayCanvas Editor の基本的な機能を使って簡単なゲームを作成します。こちらは GUI の操作となっており。プログラミングのコードと 3D モデルにつきましては PlayCanvas Editor というものを使用してすでにアセットが入った状態で、PlayCanvas の操作を学びながら作成をしていきます。

2 章では PWA(Progressive Web App)と言われるものや Firebase や CircleCI といったウェブ開発で使われているツールを使用した使ったデプロイ方法など、実際に開発したゲームを自分のサイトで公開する方法紹介します。

この本を読んだ後、Web で既にあるゲーム資産の活用を考えたり、PlayCanvas を使用して Web ゲームを作りたいと思う方が一人でも増えていただけると嬉しいです。
